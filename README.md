# brunch-static-tools

A Brunch plugin that wraps [static-tools](https://github.com/jkriss/static-tools).

`brunch build --production` will automatically run the static-tools processes.

If you use [brunch-pure-static](https://github.com/jkriss/brunch-pure-static), the
server will work with the processed files (e.g. `brunch watch --production --server`).
